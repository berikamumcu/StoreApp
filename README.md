# StoreApp – React Alışveriş Sepeti

React kullanılarak geliştirilmiş basit bir e-ticaret alışveriş sepeti uygulamasıdır.

##  Özellikler
- Ürün listeleme
- Sepete ürün ekleme
- Aynı ürün tekrar eklenirse adet artırma
- Sepet toplam tutarını otomatik hesaplama
- Bootstrap ile responsive tasarım

## Kullanılan Teknolojiler
- React
- JavaScript (ES6)
- Bootstrap
- useState Hook

##  Kurulum
```bash
npm install
npm run dev
